# 「二箍輾轉」 #

吸K嗨SCHOOL的3D列印車車「二箍輾轉」 for 2018年第一屆3D列印工程設計創作競賽

### 洋名 ###
Dual Wheel Roller

### 檔案說明 ###
- 根目錄有Arduino專案檔(Kart_gen3_ForCompetition.ino)
- Dual Wheel Roller資料夾存放Unity專案
- KartGen3放著Android Studio的檔案 >~<
- Builds資料夾有可安裝的apk檔
